# Quillwork

A full-site-editing WordPress theme for writers, editors, and personal brands who care about how type looks on the page.

Quillwork is part of the [This Is My URL](https://thisismyurl.com) free theme line, built on the [Colophon core](https://github.com/thisismyurl/thisismyurl-colophon).

---

## What it is

A literary, editorial theme. The design choices are deliberate:

- **Cormorant Garamond** for display headlines with italic accents — a typeface built for long-form reading
- **Newsreader** for the body column — comfortable at 400 words or 4,000
- Deep teal and warm ochre on a cream ground — calm, not stark
- Numbered service cards, gentle float accents, a portfolio layout that gets out of the way of the work
- Reduced-motion respected throughout; the animations that exist are subtle and defeatable

All three font families are self-hosted OFL files — no Google Fonts request, no third-party connection on page load.

## Install

**Option A — from the WordPress.org directory (recommended):**

1. WordPress admin → Appearance → Themes → Add New
2. Search for **Quillwork**
3. Install and activate

Or install directly: [wordpress.org/themes/quillwork](https://wordpress.org/themes/quillwork/)

**Option B — from a release ZIP:**

1. Download the latest ZIP from [Releases](../../releases)
2. WordPress admin → Appearance → Themes → Add New → Upload Theme
3. Upload the ZIP, activate

**Option C — clone (development):**

```bash
cd wp-content/themes/
git clone https://github.com/thisismyurl/quillwork quillwork
```

Then activate via Appearance → Themes. Note: cloning `main` gives you the development branch, which may be ahead of or behind the current WordPress.org release.

## What you get

- WCAG 2.2 AA accessibility — skip link, visible focus rings, reduced-motion support, ARIA patterns — cleared by a defensible margin, not just passing a checker
- Core Web Vitals discipline — no render-blocking scripts, no layout shift from late-loading fonts, no third-party connections
- Zero front-end JavaScript — none
- Full RTL support via CSS logical properties
- Full-site editor ready — edit headers, footers, and templates without touching code
- Block patterns for common layout needs — editorial grid, portfolio row, service cards
